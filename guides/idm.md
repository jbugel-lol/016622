## Download and Activation Instructions for Internet Download Manager (IDM)

**Before you begin:**

- Freshly install [Internet Download Manager](https://www.internetdownloadmanager.com/).
- Make sure to remove any previous cracks or patches if they exist.

### Method 1 - PowerShell (Recommended)

1. On Windows 8.1/10/11, right-click on the Windows start menu and select PowerShell or Terminal (Not CMD).
2. Copy and paste the following code and press Enter:
   ```powershell
   irm https://massgrave.dev/ias | iex
   ```
3. Follow the on-screen instructions for activation.

### Method 2 - Traditional

1. Download the file from [GitHub](https://github.com/WindowsAddict/IDM-Activation-Script) or [Bitbucket](https://bitbucket.org/WindowsAddict/idm-activation-script/).
2. Right-click on the downloaded zip file and extract it.
3. In the extracted folder, run the file named `IAS.cmd`.
4. Follow the on-screen instructions for activation.

### Activation Information

- This script uses the registry lock method to activate Internet Download Manager (IDM).
- Internet access is required during activation.
- You can install IDM updates directly without needing to reactivate it.
- If IDM starts showing an activation nag screen after activation, simply run the activation option again without using the reset option.

### Reset IDM Activation / Trial

- IDM offers a 30-day trial period, and you can use this script to reset the activation/trial period whenever needed.
- This option can also be used to restore status if IDM reports a fake serial key or similar errors.

### OS Requirements

- This project is supported on Windows 7/8/8.1/10/11 and their Server equivalents.
- The PowerShell method to run IAS is supported on Windows 8 and higher.
